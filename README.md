## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Sections

- **Hero**: Introduction with name and title
- **Projects**: Showcase of featured projects
- **Skills**: Grid of technologies and tools
- **Contact**: Links to email, GitHub, and LinkedIn

## Customization

1. Update personal information in `components/Hero.tsx`
2. Add your projects in `components/Projects.tsx`
3. Modify skills list in `components/Skills.tsx`
4. Update contact links in `components/Contact.tsx`

## Build for Production

```bash
npm run build
npm start
```
