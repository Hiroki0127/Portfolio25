## Image Guidelines:

- **Format**: PNG or JPG
- **Thumbnail size**: Recommended 1200x630px (16:9 ratio)
- **Screenshot size**: Recommended 1920x1080px or original app resolution
- **File size**: Keep under 500KB for fast loading (use compression tools if needed)

## Adding Demo Videos:

For demo videos, you can:
1. Upload to YouTube and use the embed URL
2. Add the YouTube URL to the `demoVideo` field in `/lib/projectsData.ts`
